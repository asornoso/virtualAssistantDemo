What is this?

"Okay Demo" is a simple virtual assistant created in vanilla javascript and web API's. Web Speech Recognition and Web Speech Synthesis API's are the underlying technologies that make this project work. These API's are wrapped in a voice wrapper class which is utilized by a Virtual Assistant Class which simplifies the creation of your virtual assistant, and addition of commands and actions.

How do I use it?

To use this assistant you must first click the "play" button to enable the assistant. Secondly, you must use the activation pharse, "okay demo" to awaken Demo. After that you must state your command. At this time Demo can only fetch dog photos, fetch dog photos, play fortnite, and tell jokes.

The center "stop" button will end all voice synthesis the browser has queued.

The pause button will stop the virtual assistant.

How does it work?

To expand upon the "What is this?" description, the underlying tech used are the Web Speech Recognition and Synthesis API's that are wrapped into ES6 Javascript classes for easier usage. The virtual assistant exists in states. Depending on the state, we compare the user input to the activation command, or the commands given to the assistant. We use event listeners to keep everything asynchronous.
